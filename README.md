## Лаба 2 (Проект и т.п.)
Для сдачи лабы я выбрал сделать сайт для компьютерной игры, которую разрабатываю в свободное время с командой. Сейт для игры нужно было сделать так или иначе и это удачный случай для этого. Основным требованием для сайта является
- лаконичноость
- наличие "сочной" кнопки для подписки на вишлисты
- "игровое" настроение сайта (сайт для игры должен вводить человека в игру, в которую ему предлагается сыглать)
- наглядные call-to-action (все основные маркетинговые действия должны быть на виду aka на главной)

## Лаба 4 (Более продвинутая работа с CSS/JS)
Задание на 4 лабораторную работу предлогало мне использовать некоторый более продвинутый функционал CSS / JS. Вот некоторые задачи которые мне пришлось выполнить, чтобы улучшить свой проект и попрактиковаться с новыми инструментами.
- Рутина: задания касающиеся flexbox и grid были для меня по сути либо выполнены на этапе второй лабы, либо были трансформацией неудачных (как мне кажется) решений в более удачные. Также необходимо было пользоваться переменными css для обозначения цветов. Я сделал это чисто для задания, хотя в моем случае это было явно излишеством (единственные цвета, которые я использовал были черный, белый и прозрачный -- цвета для которых УЖЕ введены ключевые слова...). 
- Интересной оказалась моя работа с анимацией. Я решил реализовать простенький рендер диалога (побуквенный) и твин появления диалогового окна. Первое я реализовал с использованием css анимации, второе -- используя асинхронность js. На момент реализации я не совсем знал о том, как устроены изнутри промисы и как устроен цикл обработки js и делал все это на интуиции того, как асинхронность и корутины работают в знакомом мне c#. Однако, сейчас узнав об этом чуть больше я бы, наверное вложил чуть больше усилий для того, чтобы работа в функциях с async / await проходила более гладко (мой вариант был скорее наивным решением).
- Во второй лабе страницу с формой я делал используя position: relative. Это решение было явно неудачным. В этой лабе я переделал все на position: absolute, довел форму до ума и сделал минимальную обертку формы в объект js.
